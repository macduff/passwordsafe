Вкладка "Отображение"

Параметры на вкладке Отображение управляют отображением Password Safe и его элементов.

Отображать Password Safe поверх всех окон

Если помечено, развернутый Password Safe будет находиться поверх всех окон, независимо от того, какое окно имеет фокус ввода.

Отображать имя пользователя в дереве

Если помечено, имя пользователя будет отображаться в квадратных скобках за именем элемента (в режиме древовидного списка).

Отображать пароль в дереве

Если помечено, пароль будет отображаться в фигурных скобках после имени пользователя (в режиме древовидного списка).
Данный параметр строго не рекомендуется использовать, если во время работы возможен просмотр экрана посторонними людьми.

Отображать заметки во всплывающей подсказке

Если помечено, заметки будут отображаться при наведении курсора мыши на элемент.

Отображать пароль при добавлении и редактировании

Если помечено, скрытый по умолчанию пароль в диалогах добавления и просмотр/редактирования будет видимым. Вы можете отобразить или скрыть пароль текущего элемента, нажав в диалоге кнопку "Показать/Скрыть".

Отображать заметки при редактировании

Если помечено, в диалоге редактирования заметки будут видимыми по умолчанию. В противном случае, нужно щелкнуть в поле заметок для отображения.

При редактировании переносить заметки по словам

Если помечено, в поле заметок будет использоваться перенос по словам.
Иначе, будет отображаться горизонтальная полоса прокрутки.

Отображать сетку списка

Если помечено, будут видны светлые линии сетки, и список будет похож на таблицу.

Отображать группы вверху дерева

В древовидном списке по умолчанию элементы отображаются в алфавитном порядке. Если данная опция активна, группы будут расположены вверху (в алфавитном порядке), а за ними пойдут остальные элементы соответствующего уровня.

Подсветить измененные элементы

Пока элемент не сохранен, он будет помечен цветом. Если на вкладке "Резервные копии" активна опция "Сохранять контейнер после вставки или редактирования", элементы никогда не будут помечены, так как сохранение происходит немедленно.

Предупреждать за Nдней до истечения пароля

Если помечено, при чтении контейнера (обычно при старте) Password Safe будет просматривать все элементы и уведомлять, если пароль истекает не позднее чем через N дней от текущей даты.

Вид дерева при старте

Определяет, как отображать элементы при открытии контейнера в режиме древовидного списка.

Значок в области уведомлений

Позволяет указать цвет значка Password Safe в области уведомлений (трее), когда контейнер еще не открыт.
