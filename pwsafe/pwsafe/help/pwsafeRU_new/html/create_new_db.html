Создание и открытие нового контейнера

Password Safe позволяет хранить все пароли в одном или нескольких (один для рабочих записей, другой для личных) контейнерах. Контейнеры являются независимыми и могут перемещаться и использоваться на различных машинах, в случае, если версии Password Safe совпадают. Для шифрования контейнеров используется ключ, полученный на основе мастер-пароля (сам мастер-пароль не содержится в контейнере ни в каком виде).

Если вы используете Password Safe впервые, нажмите кнопку "Новый контейнер". Вам будет предложено указать каталог и имя для нового контейнера. После этого вам предложат ввести мастер-пароль, который будет использован для шифрования контейнера.

Создание нового контейнера

Контейнер можно создать двумя способами. Щелкнув Новый контейнер в диалоге Пароль для контейнера, выводимом при запуске программы, или из меню Файл > Новый контейнер, которое доступно когда Password Safe уже открыт. В любом из этих случаев будет выведен диалог Установка пароля для контейнера.

Примечание:по умолчанию, Password Safe отображает последний использованный контейнер и запрашивает пароль для него. Если вы используете несколько контейнеров, можно выбрать другой контейнер из выпадающего списка. Также можно ввести путь к контейнеру или выбрать его нажав на кнопку "...". Если контейнер уже открыт, вы можете открыть другой, нажав Открыть контейнер в меню Файл.



Экранная клавиатура

Внутри мастер-пароля вы можете использовать символы иностранных алфавитов, например, ä, ö для немецкой раскладки. Чтобы ввести такие символы вам потребуется соответствующая раскладка. Ее можно получить, используя экранную клавиатуру. Данная клавиатура может быть использована для всех иностранных алфавитов. Для отображения экранной клавиатуры нужно нажать символ . Указанный метод может использоваться для ввода символов, которые недоступны при использовании системной раскладки. Таким образом можно ввести желаемый мастер-пароль. Экранная клавиатура позволяет вводить даже части паролей.
