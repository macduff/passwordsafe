Вкладка "Разное"


На вкладке Разное настраиваются раличные действия по умолчанию.

Подтверждать удаление элементов

Если помечено, при удалении элемента будет выводится запрос подтверждения.

Сохранять время последнего доступа к записи

Если помечено, Password Safe будет записывать время доступа к элементу (при копировании пароля, просмотре данных элемента и пр.). Время хранится вместе с элементом и может быть просмотрено в диалоге редактирования/просмотра элемента.
Примечания:
  • В данном случае контейнер будет изменяться даже если элементы не редактировались. Это значит, что при закрытии контейнера вам будет предложено сохранить изменения, даже если вы ничего явно не изменяли.
  • В случае, если контейнер открыт в режиме "только для чтения", время последнего доступа не обновляется.

Закрывать приложение по Esc

Если помечено, клавиша Esc закрывает Password Safe. Заметьте, что если активна настройка "Отображать значок в области уведомлений", то клавиша Esc лишь скрывает окно, которое может быть открыто заново двойным щелчком по значку в области уведомлений.

Вызов по

Если помечено, указанная комбинация клавиш будет использована для восстановления Password Safe из скрытого состояния, или помещения его окна на передний план, если оно перекрыто другими окнами. Комбинация клавиш может быть изменена при установке флажка настройки.

Действие по двойному щелчку

Позволяет установить действие, которое будет выполнять Password Safe при двойном щелчке на элементе. Действием по умолчанию является копирование пароля в буфер обмена. Возможные следующие действия:

  • Автонабор
  • Перейти по ссылке
  • Перейти по ссылке и выполнить автонабор
  • Копировать заметки в буфер обмена
  • Копировать пароль в буфер обмена
  • Копировать пароль в буфер и свернуть
  • Копировать имя пользователя в буфер
  • Выполнить команду
  • Отправить письмо
  • Просмотр/редактирование элемента

Обратите внимание, что поведение "Копировать пароль в буфер и свернуть" будет не таким, как ожидалось, если включена настройка "Очищать буфер обмена при сворачивании" на вкладке Безопасность.

Автонабор

  • Свернуть перед автонабором По умолчанию, Password Safe сворачивает окно перед выполнение автонабора. Если флажок снят, вместо сворачивания Password Safe будет перемешать свое окна на задний план.
  • Строка автонабора по умолчанию Здесь можно указать строку автонабора, применяемую по умолчанию. При необходимости вы можете переопределить ее в настройках элемента. Например, данную настройку можно использовать для изменения задержки автонабора для всех элементов. Для получения дополнительной информации об автонаборе см. здесь.

Имя пользователя по умолчанию

Данная настройка позволяет указать имя пользователя, подставляемое по умолчанию при создании нового элемента. Если помечено, указанное имя будет подставляться в новые элементы по умолчанию. При создании элемента вы можете изменить это имя, если потребуется.

Альтернативный браузер

Указанный здесь браузер будет использован при выполнении команды "Перейти по ссылке", если в настройках элемента сыылка начинается с "[alt]", например, "[alt]http://www.foo.net". Данный функционал может потребоваться, например, если в качестве браузера по умолчанию настроен Firefox, а связанный с элементом сайт лучше работает в Internet Explorer. В данном случае необходимо указать Internet Explorer как альтернативный браузер ("C:\Program Files\Internet Explorer\iexplorer.exe").
Параметры командной строки для браузера указываются в поле "параметры командной строки".
