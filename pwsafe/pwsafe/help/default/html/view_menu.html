View Menu

The View Menu controls the available display formats. You can switch between Classic and New Toolbar Icons as well as how their user names and passwords are organized: either as a single list (Flattened List) or organized by category (Nested Tree). You can also expand or collapse the groups displayed in the Nested Tree view, as well as select the font in which the entries are displayed.

Switching Between Menu Icon Views

  1. Go the the View menu
  2. Select Classic Toolbar or New Toolbar

Switching Between Nested Tree and Flattened List Views

  1. Go the the View menu
  2. Select Nested Tree or Flattened List

Expand All

Expands all groups, making all entries visible (Nested Tree view only).

Collapse All

Collapses all groups (Nested Tree view only).

Customize Main Toolbar

Define which buttons to display on the toolbar, and in what order. See here for a list of the buttons and their meaning.

Change Font

Select a different font for displaying entries.

Reports

Opens default text editor (e.g., Notepad) on the results of merge, compare, import and validate operations.
