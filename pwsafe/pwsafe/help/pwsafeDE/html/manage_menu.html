Menü Verwalten

Das Menü Verwalten verwaltet sämtliche Optionen und Einstellungen von Password Safe.

Master-Passwort ändern

Bei der Auswahl dieses Reiters erscheint der Dialog Ändere das Master-Passwort.
Siehe Master-Passwörter ändern für mehr Information.

Sicherung erstellen

Erstellt eine Sicherung von der aktuellen Datenbank.
Siehe Datenbasen sichern und wiederherstellen für mehr Information.

Wiederherstellen aus Sicherungen

Die Datenbank wird aus einer Sicherung wiederhergestellt.
Siehe Datenbasen sichern und wiederherstellen für mehr Information.

Optionen...

Bei der Auswahl von Optionen... erscheint ein Dialog mit mehreren Reitern. Viele der Einstellungen in Password Safe können hier eingesehen und bearbeitet werden. Um mehr über einen bestimmten Reiter zu erfahren, wählen Sie ihn unten aus:

Zwischen-Sicherungen
Anzeige
Sicherheit
System
Passwortrichtlinie
Passworthistorie
Verschiedenes
Schnelltasten
Hinweis:
Viele dieser Einstellungen werden als Voreinstellungen abgespeichert, wie hier beschrieben.

Toolleiste anpassen

Wählen Sie die Symbole aus die in der Toolleiste dargestellt werden sollen (oder auch nicht). Die Reihenfolge der Symbole kann dabei umgestaltet werden.
