Upgraden einer älteren Version von Password Safe

Um neue Merkmale zu unterstützen und die Sicherheit zu erhöhen wurde die Datenbank für die Gespeicherten Informationen und Einstellungen zwischen zwei Versionen geändert. Es wurde aber größte Sorgfalt aufgebracht um eine automatisches und transparentes Upgraden zu ermöglichen:
  • Version 1.x von Password Safe kann nur Datenbanken lesen die von Version 1.x generiert wurden.
  • Version 2.x von Password Safe kann Datenbanken lesen die aus den Versionen 1.x und 2.x stammen.
  • Version 3.x von Password Safe kann Datenbanken lesen die aus allen vorherigen Versionen stammen.
Um die Datenbank einer früheren Version von Password Safe zu öffnen, sollten sie sie auf jeden Fall so öffnen wie jede andere Datenbank auch. Die Einträge werden angezeigt wie immer, mit möglicherweise "unbekannte" Werte in Felder die in den früheren Versionen nicht unterstützt wurden (z.B. das Erstellungsdatum eines Eintrages).
Wenn sie eine Datenbank aus einer früheren Version abspeichern, werden sie gewarnt über die Tatsache dass die neue (abgespeicherte) Datenbank von früheren Versionen von Password Safe nicht mehr geöffnet werden kann.

Frühere Versionen der Datenbank generieren

Sie können Versionen der Datenbanken erstellen die von früheren Versionen von Password Safe gelesen werden können, durch gezieltes Exportieren im gewünschten Format (Datei->Exportiere zu und wähle V1.x Format... oder V2 Format... aus). Beachte dass in der exportierten Datenbank keine Informationen enthalten sind die in neuere Versionen hinzugekommen sind. Zum Beispiel, eine Version 2 der Datenbank wird keine Informationen über die Änderungszeitpunkte von Einträgen enthalten.
