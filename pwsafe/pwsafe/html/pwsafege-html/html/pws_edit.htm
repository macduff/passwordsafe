Datenbank-Eintrag bearbeiten/anzeigen

In diesem Fenster (Titel "Edit/View Entry") können Sie die zu einem Eintrag gehörenden Angaben ändern. Standardmäßig wird das Passwort nicht im Klartext anzeigt, sondern durch Sternchen ('*****') verdeckt. Um das Passwort im Klartext anzuzeigen, klicken Sie auf "Show Password". (Sie können auch alternativ die Option "Show password by default in edit mode" aktivieren.)

Für eine Beschreibung der einzelnen Eingabefelder schauen Sie bitte unter dem Hilfethema "Datenbank-Eintrag hinzufügen" nach.

Haben Sie bei der Eingabe einen Fehler gemacht und wollen Sie das Fenster ohne zu speichern verlassen, klicken Sie auf "Cancel". Um die Änderungen zu speichern, klicken Sie auf "OK".
