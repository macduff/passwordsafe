Datenbank-Passwort ändern

Im Fenster "Key Change" können Sie das Passwort für die momentan geöffnete Datenbank ändern. Zunächst müssen Sie das alte Passwort einmal eingeben, danach das neue Passwort zweimal. Die Eingabe wird mit Sternchen verdeckt. Das Passwort ist sensibel für Groß- und Kleinschreibung, d.h. "hallo" ist nicht das gleiche wie "HALLO". Zum Abschluss klicken Sie auf "OK". Der "OK"-Button wird nur dann aktiviert sein, wenn Sie das alte Passwort sowie das neue Passwort (zweimal) korrekt eingegeben haben.
