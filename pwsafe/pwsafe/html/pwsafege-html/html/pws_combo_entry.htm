Datenbank-Passwort eingeben

Das Fenster "Safe Combination Entry" sehen Sie immer dann, wenn Sie Password Safe starten, wenn Sie eine neue Datenbank öffnen oder wenn Sie eine Datenbank von einer Sicherheitskopie wiederherstellen. Geben Sie das Passwort für die Datenbank ein und klicken Sie auf "OK". Das Passwort ist sensibel für Groß- und Kleinschreibung, d.h. "hallo" ist nicht das gleiche wie "HALLO".

Falls das Passwort dreimal falsch eingegeben wurde, erscheint das "Incorrect Passkey"-Fenster. Sie können hiermit eine andere Datenbank öffnen oder eine neue erstellen.
