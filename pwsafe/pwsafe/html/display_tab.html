Display Tab

Functions within the Display Tab control how the Password Safe GUI and password items are displayed.

Always keep Password Safe on top

If checked, maximized Password Safe will always be the top most window displayed in the montior, regardless of what other window may have focus.

Show password in dislay list

If checked, passwords will be displayed in a colomun next to their associated user name when viewed in Flattened List Mode.
This option is strongly discouraged in environments where passers-by can view the montior.

Show password by default in edit mode

If checked, passwords that would normally be obscured in the Edit/View Entry dialog will be displayed in plain text.

Put Icon in System Tray

If checked, Password Safe will minimize to the the System Tray area. If unchecked, the program will minimize to the task bar.
