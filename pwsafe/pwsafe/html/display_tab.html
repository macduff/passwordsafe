Display Tab

Functions within the Display Tab control how Password Safe and password items are displayed.

Always keep Password Safe on top

If checked, maximized Password Safe will always be the top most window displayed in the montior, regardless of what other window may have focus.

Show password in display list & tree

If checked, passwords will be displayed in a column next to their associated user name when viewed in Flattened List Mode, or in square brackets following the user's name in the Nested Tree Mode view.
This option is strongly discouraged in environments where passers-by can view the montior.

Show password in add & edit mode

If checked, passwords that would be hidden in the Add Entry and Edit/View Entry dialogs will be displayed in plain text. You can display the passwords by clicking on the "Show" button in the dialog.

Show notes in edit

If checked, notes will be displayed by default in edit dialog. If not, you will need to click on the Notes field in order to view the text there.

Initial tree view

This determines how your entries will be displayed in the tree view when you open the database.

Put icon in System Tray

If checked, Password Safe will minimize to the the System Tray area. If unchecked, the program will minimize to the task bar.

The icon in the system tray indicates whether the password database is locked () or unlocked (). If it is locked, you will be prompted for the master combination when you try to unminimize it.

Remember last used entries in System Tray menu

If the "Put icon in System Tray" option is enabled, you can right-clisk on the system tray icon to get to the most recently used entries directly, without going through the application's main screen. This option allows you to configure how many recent entries you wish to keep.

Number of Recent Entries kept in history

If the Put Icon in System Tray option is enabled, right-clicking on the Password Safe icon in the System Tray will open a menu that will allow you to rapidly access the last entries you've used since starting up Password Safe. Here you can configure how many entries to save, up to a maximum of 25.

Recent Passwordsafe Databases

These controls allow you to configure how many recently opened databases are displayed in the File menu, for easy access. Note that you need to exit and restart Password Safe for changes in this control to take effect.

Recent Databases on File Menu rather than as a sub-menu

This option determines how the recently opened databases are presented in the File menu - directly on the menu, or as a sub-menu.
