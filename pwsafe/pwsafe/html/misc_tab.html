Misc Tab


Various miscellaneous program defaults are stored in the Misc Tab.

Confirm deletion of items

When checked, a user prompt requesting confirmation will be displayed every time a user tries to delete a password entry.

Save database immediately after edit or add

When checked, Password Safe saves all changes made to a password entry immediately, without waiting for the user to use Save or Save As and without confirmation.

Escape key exits application

When checked, Esc can be used to exit Password Safe when the program is maximized.

Hot Key

When checked, pressing the keys in the sequence to the right will restore PasswordSafe from the system tray, if minimized, or bring it to thefront, if obscured by other applications. The sequence may be changed when the option is checked.

Double click on entry

This allows you to define what action PasswordSafe will take when you double-click on an entry. The default is to copy the entry's password to the clipboard.
