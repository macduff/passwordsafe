Using Stored Passwords

Password Safe provides several mechanisms for using stored user names and passwords. Most require copying the user name or password to the clipboard and then pasting the information into the required input field. One function that automates the entering of user name and password into a web form is covered elsewhere in this manual. See Auto Type for more information.

  1. Open Password Safe
  2. Select the user name/password entry you wish to use
    User Name:
  • Select the user name icon from the tool bar
    or
  • Right-click and select Copy username to clipboard
    or
  • Use Ctrl + U
    and then Paste the user name into the desired field.
    Password:
  • Select the password icon from the tool bar
    or
  • Right-click and select Copy password to clipboard
    or
  • Double-click the selected entry

    • or
  • Use Ctrl + C
    and then Paste the password into the desired field.
Note: User names and passwords can also be copied to clipboard through the choices located in the Edit menu, but let's be honest, that's far too much clicking.
