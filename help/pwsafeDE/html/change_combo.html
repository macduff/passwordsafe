General-Passwörter ändern

Um ein General-Passwort ("Safe Combination") zu ändern, muss man zuerst die Datenbank wovon das Passwort geändert werden soll, geöffnet werden. Einmal geöffnet, gehe zum Menü Verwalten und wähle General-Passwort ändern aus. Sie sollten das derzeitige General-Passwort eingeben, und dann das neue General-Passwort (zweimal, zur Bestätigung).

Password Safe hat eine eingebaute Prüfung auf vermeintlich schwache Passwörter. Im Allgemeinen sollte ein General-Passwort (wie jedes andere Passwort auch) aus einem Mix von Buchstaben und mindestens eine Ziffer oder Spezielle Zeichen (wie diese über die Ziffern an der Tastatur). Wenn das Programm ein eingegebenes General-Passwort als schwach einstuft, wird es um eine Bestätigung fragen. Sie haben dann die Wahl das original eingegebene (schwache) General-Passwort anzunehmen, oder ein Neues einzugeben.

Notes:
  • Bemerkung: Es besteht keine Möglichkeit in Password Safe verlorene oder vergessene Passwörter wieder aufzufinden.
    Dies wurde im Konzept bewusst so entschieden, weil jede Art Passwörter wieder herzustellen auch eine Sicherheitslücke bedeuten würde.
  • Wählen sie bitte ein schwer zu erratendes Passwort. Ein triviales Passwort wie "passwort" oder Ihr Name wäre einfach zu erraten und würde damit die Sicherheit die Password Safe bietet zuwiderlaufen.
