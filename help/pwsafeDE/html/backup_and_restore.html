Datenbanken sichern und wiederherstellen

Für gewöhnlich sind die Informationen die in Passwortdatenbanken enthalten sind sowohl sensibel als auch wichtig. Password Safe stellt einige Mechanismen zur Verfügung um Passwörter zu sichern und wiederherzustellen. Diese Kapitel beschreibt Wege um händisch Sicherungen ihrer Passwortdatenbanken zu machen. Password Safe kann auch so eingestellt werden dass automatisch Sicherungen gemacht werden, wenn die Datenbank gespeichert wird. Solche Sicherungskopien werden Zwischen-Sicherung genannt, und werden hier beschrieben.

Sicherung erstellen

Erstelle eine Kopie der aktuellen Passwortdatenbank.

  1. Klick Sicherung erstellen
  2. Wähle eine Speichestelle für die Sicherung
  3. Klick Speichern

Hinweis: es wird dringend empfohlen regelmäßig Sicherungen der Passwortdatenbanken zu machen. Dies schützt Passwortlisten gegen Systemzusammenbrücke, Stromunterbrechungen und kaputte Dateien. Die Sicherung ist genau so wie die Datenbank verschlüsselt, mit demselben General-Passwort.

Wiederherstellen aus Sicherung

Die Wiederherstellung einer Passwortdatenbank aus einer Sicherung.

  1. Klicke Wiederherstellung aus Sicherung an
  2. Gehe zum Verzeichnis wo die Sicherung sich befindet
  3. Wähle die entsprechende Erweiterung (.bak/.ibak) aus
  4. Wähle die gewünschte Datei aus
  5. Klick Öffnen
    Ein Aufforderung zur Eingabe des Passwortes wird angezeigt.
  6. Gib das Passwort ein und klick OK
Hinweis:
  • Die wiederhergestellte Datenbank ist solange nicht "gespeichert" mit der Password Safe Standarderweiterung (.psafe3) bis sie nicht Speichern oder Speichern unter im Menü Datei ausgeführt haben. Wenn sie die Anwendung verlassen wollen ohne die Datenbank abgespeichert zu haben, wenn sie dazu vorher aufgefordert dies zu tun.
  • Zusätzlich zum Erstellen und Wiederherstellen wie es hier beschrieben ist, wird Password Safe automatisch eine Kopie der Datenbank vorhalten, bevor Änderungen abgespeichert werden. Diese Kopie wird im selben Verzeichnis wie die aktuelle Datenbank abgelegt. Sie hat den selben Namen aber mit anderer Erweiterung (".ibak" anstelle von ".psafe3"). Siehe hier für mehr Information über automatische Sicherungen.
