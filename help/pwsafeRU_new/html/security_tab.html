Вкладка Безопасность

На данной вкладке можно изменить пользовательские настройки безопасности. Указанные настройки влияют на безопасность и простоту работы с программой, не не затрагивают сам контейнер.

Очищать буфер обмена при сворачивании

Если помечено, при сворачивании окна программы будет произведена очистка буфера от данных, помещенных туда Password Safe (например, имя пользователя, пароль и пр.). Таким образом никто не сможет прочитать то, что было записано в буфер. Обратите внимание, что использование данной опции вместе с "Свернуть после копирования пароля" на вкладке Разное приведет к неожиданным результатам (пароль будет скопирован в буфер и сразу же удален оттуда).

Очищать буфер обмена при выходе

Если помечено, при закрытии окна программы будет произведена очистка буфера от данных, помещенных туда Password Safe (например, имя пользователя, пароль и пр.). Таким образом никто не сможет прочитать то, что было записано в буфер.

Подтверждать копирование элемента в буфер обмена

Если помечено, Password Safe каждый раз будет запрашивать разрешение на копирование пароля в буфер обмена.

Блокировать контейнер при сворачивании

Если помечено, при сворачивании программы контейнер будет блокироваться. Для доступа к контейнеру после распахивания окна нужно будет ввести мастер-пароль.

Блокировать контейнер при блокировке компьютера

Если данная настройка включена, контейнер будет блокироваться при блокировке компьютера. Ее можно использовать для предотвращения несанкционированного доступа к контейнеру лиц, имеющих доступ к рабочей станции.

Блокировать контейнер после...

Если включено, Password Safe будет блокировать контейнер по истечении указанного времени с момента выполнения последнего действия с программой.
