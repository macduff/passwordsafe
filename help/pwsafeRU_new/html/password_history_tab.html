Управление историей паролей



Password Safe может сохранять предыдущие пароли при изменении пароля элемента. Это может пригодится в том случае, когда смена пароля на соответствующем сайте завершается ошибкой. Данная настройка указывает, нужно ли по умолчанию включать сохранение предыдущих паролей для новых элементов. Данную настроку можно переопределить в диалоге добавления или редактирования элемента.

Сохранять N предыдущих паролей для элемента

Установка данного флажка задает количество паролей, сохраняемых для новых элементов по умолчанию.

Управление историей паролей для текущих элементов

Данный раздел позволяет управлять настройками хранения паролей для уже созданных элементов:
  • Без изменений - настройки сохранения предыдущих паролей не изменяются
  • Перестать сохранять предыдущие пароли - это равносильно снятию флажка "Сохранять N предыдущих паролей" для всех элементов контейнера. Обратите внимание, что существующая история будет потеряна и изменения паролей в этих элементах не будут отслеживаться до момента повторного включения.
  • Начать сохранять предыдущие пароли - это равносильно установке флажка "Сохранять N предыдущих паролей" для всех элементов контейнера. Элементы, история для которых не сохранялась, будут запоминать указанное количество предыдущих паролей. Элементы, ведущие историю паролей, продолжат сохранять изменения, при этом количество сохраняемых паролей не будет изменено.
  • Установить максимальное количество сохраняемых паролей равным указанному значению - все элементы будут вести хранить указанное количество паролей. Но, если ранее у каких-либо элементов было указано большее значение, история не будет потеряна.
