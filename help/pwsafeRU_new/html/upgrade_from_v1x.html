Переход на новые версии Password Safe

При изменении старшей цифры в версии программы изменяется формат контейнера. Это делается для того, чтобы повысить безопасность и/или реализовать работу новых функции. Но приложено много усилий для того, чтобы сделать это переход как можно более простым и прозрачным.
  • Версия Password Safe 1.x поддерживает чтение контейнеров, созданных в версии 1.x.
  • Версия Password Safe 2.x может читать контейнеры, созданные в версии 2.x и 1.x.
  • Версия Password Safe 3.x может читать контейнеры всех предыдущих форматов.
Открытие контейнера, созданного с помощью предыдущих версий Password Safe происходит таким же образом, как открытие контейнера для текущей версии. Элементы будут иметь тот же вид, за исключением того, что некоторые поля могут быть не заполнены, так как они отсутствовали в ранних версиях (например, дата создания элемента).
При сохранении такой контейнер будет обновлен до текущего формата, и перестанет распознаваться предыдущими версиями Password Safe.

Создание контейнеров в формате предыдущих версий

Вы можете создать копии контейнера, читаемые в предыдущих версиях Password Safe используя экспорт в требуемый формат (Файл->Экспортировать в и выберите формат v1.x... или формат v2.x...). Обратите внимание, что в этих контейнерах не будет содержаться информации, добавленной в новых версиях. Например, контейнер второй версии не будет содержать данных о времени изменения элемента.
