Меню "Изменить"

Элемент Алиас Группа

Используя меню Изменить вы можете добавлять, редактировать, использовать и упорядочивать имена пользователей и пароли. Обратите внимание, что состав меню зависит от того, что сейчас выбрано в списке (элемент, алиас или группа). Кроме того как указано ниже, некоторые элементы могут быть недоступны.

Добавить элемент

Добавление нового имени и пароля. Недоступно, если контейнер открыть в режиме "только для чтения".
Для получения дополнительной информации см. имена пользователей, пароли и группы.

Редактировать элемент / Развернуть/свернуть группу

Если выбран элемент, открывает диалог редактирования текущей записи. Если выбрана группа (при древовидном просмотре) сворачивает или разворачивает список элементов для текущей группы. Редактирование заблокировано, если контейнер открыт в режиме "только чтение".
Для получения дополнительной информации см. имена пользователей, пароли и группы.

Удалить элемент / Удалить группу

Если выбрана для элемента, удаляет его. Если выбрано для группы (при древовидном просмотре), будут удалены все элементы, входящие в группу. При удалении группы из нескольких элементов будет выдан запрос подтверждения. Пункт заблокирован, если контейнер открыт в режиме "только для чтения".

Переименовать элемент / Переименовать группу

Изменяет имя для элемента или группы (при древовидном просмотре). Пункт заблокирован, если контейнер открыт в режиме "только для чтения".

Найти элемент

Поиск элемента, содержащего указанный текст. По умолчанию поиск производится по всем полям, кроме пароля.
Для получения дополнительной информации см. Поиск.

Дублировать элемент

Дублирует выбранный элемент. Пункт заблокирован, если контейнер открыт в режиме "только для чтения".

Добавить группу

Создает новую группу (например, "Пароли для пластиковых карт"). Пункт заблокирован, если контейнер открыт в режиме "только для чтения".

Вернуть

Последнее действие будет отменено. При повторных нажатиях отмена будет производиться в обратном порядке (от последнего действия к первому).

Повторить

Последнее отмененное действие (действия) будет выполнено снова.

Очистить буфер обмена

Очищает содержимое буфера обмена. Очистка производится для любых элементов, помещенных туда Password Safe (имена, пароли, заметки, ...). Не оказывает действия, если содержимое буфера не принадлежит Password Safe.

Копировать пароль в буфер обмена

Копирует пароль выбранного элемента в буфер обмена.

Показать часть пароля

При выборе данного действия отображается окно, позволяющее указать символы, которые необходимо отобразить. Данная функция может быть полезна для сайтов, требующих ввода части пароля. Например, если задан пароль "qAdm3ddENc", вас могут попросить ввести первый, третий, пятый и восьмой символы. Это окно упрощает отбор: в верхнем поле нужно ввести позиции необходимых вам символов. Разрешается использовать в в качестве разделителя пробелы, запятые и точки с запятой. Соответствующие символы будут показаны в нижнем поле. Выход производится по кнопке Esc.

Копировать имя пользователя в буфер обмена

Копирует имя пользователя для текущего элемента в буфер обмена.

Копировать заметки в буфер обмена

Копирует содержимое заметок для текущего элемента в буфер обмена.

Копировать ссылку в буфер обмена

Копирует ссылку, связанную с текущим элементом, в буфер обмена.

Копировать E-mail в буфер обмена

Копирует E-mail, связанный с текущим элементом, в буфер обмена.

Копировать команду в буфер обмена

Копирует команду для текущего элемента в буфер обмена.

Перейти по ссылке

Если для текущего элемента указана ссылка, открывает ее в браузере по умолчанию. Если элемент не выбран, или для него не задана ссылка, команда будет заблокирована.

Перейти по ссылке и выполнить автонабор

Если для текущего элемента указана ссылка, открывает ее в браузере по умолчанию. Если элемент не выбран, или для него не задана ссылка, команда будет заблокирована.

Автоматически вводит имя пользователя и пароль в выбранные поля.
Для получения дополнительной информации см. Использование автонабора.

Отправить письмо

Использует данные из поля E-mail для создания нового письма..
Для получения дополнительной информации см. имена пользователей, пароли и группы.

Автонабор

Автоматически вводит имя пользователя и пароль в выбранные поля.
Для получения дополнительной информации см. Использование автонабора.

Run Command (выполнить команду)

Выполняет команду, указанную в поле Выполнить текущего элемента.
Для получения дополнительной информации см. Выполнение связанных команд.

Создать ярлык

Если выбран элемент, позволяет создать ссылку на этот элемент (обычно в другой группе). Ярлыки удобно использовать если элемент "относится" к нескольким группам, но вы не желаете дублировать данные. Для отображения ярлыков используется другой значок, , и, по умолчанию, имя имеет вид "ярлык для x", где x - имя исходного элемента. Как и в случае с обычным элементом, вы можете переместить, удалить или переименовать ярлык.

Перейти к исходному элементу

Данный пункт отображается только для ярлыков и алиасов. После нажатия Перейти к исходному элементу будет выбран соответствующий исходный элемент.

Редактировать исходный элемент

Данный пункт отображается только для ярлыков и алиасов. После нажатия Редактировать исходный элемент вы сможете произвести изменения в соответствующем исходном элементе для выбранного ярлыка или алиаса.

Экспортировать элемент в простой текст

Выбранный элемент будет экспортирован в виде простого текста. Для подтверждения экспорта содержимого элемента в незашифрованный файл, будет предложено ввести мастер-пароль.
Для получения дополнительной информации см. Экспорт элементов.

Экспортировать элемент в XML формат

Выбранный элемент будет экспортирован в формате XML. Для подтверждения экспорта содержимого элемента в незашифрованный xml-файл, будет предложено ввести мастер-пароль.
Для получения дополнительной информации см. Экспорт элементов.
