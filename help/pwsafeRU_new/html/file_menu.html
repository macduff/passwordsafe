Меню Файл

Через меню Файл выполняются действия, связанные с управлением и созданием контейнеров Password Safe.

Создать...

Создать новый контейнер Password Safe.

Открыть...

Открыть существующий контейнер Password Safe.

Закрыть

Закрыть текущий контейнер без завершения приложения.

Недавно использованные

Данное подменю может содержать список недавно использованных контейнеров. Подменю может быть пустым, если Password Safe запускается в первый раз.

Примечания:

  • Список недавно использованных контейнеров можно очистить нажав на пункт меню "Очистить список ранее использованных контейнеров"
  • Количество запоминаемых контейнеров настраивается на вкладке Система диалога Управление->Настройки.... Также там можно настроить отображение списка недавно используемых контейнеров в самом меню Файл, а не в подменю.

Сохранить

Сохранить изменения в текущем контейнере. Если с момента предыдущего сохранения не было изменений, или если контейнер открыт только для чтения, данный пункт меню будет заблокирован.

Сохранить как...

Сохранить текущий контейнер под другим именем или в другом каталоге.

Экспортировать в...

Экспорт содержимого открытого контейнера в другой файл в формате предыдущих версий Password Safe, XML-файл, или как простой текст. Подробная информация о сохранении в другие форматы содержится здесь.

Импортировать из...

Импортировать в текущий контейнер содержимое ранее экспортированного или правильно составленного текстового файла, XML-файла или контейнера KeePass. Подробная информация об импорте из текстовых форматов содержится здесь.

Слияние...

Слияние контейнера Password Safe или резервной копии контейнера (.bak) с текущим контейнером. Заблокировано, если контейнер открыт в режиме "только для чтения".

Сравнить...

Сравнить содержимое открытого контейнера с другим контейнером.

Синхронизировать...

Синхронизировать содержимое открытого контейнера с другим контейнером.

Основное различие между слиянием и синхронизацией состоит в том, что во время слияния все элементы из выбранного контейнера переносятся в открытый. При синхронизации происходит обновление лишь тех элементов, который присутствуют в открытом контейнере.

Примечание: при работе с большими контейнерами операции слияния, сравнения и синхронизации могут занять длительное время.

Свойства

Отображение информации о текущем контейнере: версия формата файла, количество групп, количество элементов, данные о том, когда и кем было произведено последнее сохранение.

Выход

Закрытие Password Safe. Если с момента последнего сохранения контейнер был изменен, вам будет предложено сохранить их.
