Edit Menu

Entry Group

Through the Edit Menu you can add, edit, use and organize user names and passwords. Note that the contents of this menu change according to what's selected in the main view (e.g., if you've selected and entry or a group). Also, some items may be disabled, as noted below.

Add Entry

Add a new user name and password. This is disabled if the database is opened read-only.
See User Names, Passwords, and Groups for more information.

Edit Entry / Expand/Collapse Group

When an entry is selected, Edit the selected user name and password. When a Group is selected (in Tree view), this will expand or collapse the display of entries under the selected group. Editing an entry is disabled if the database is opened read-only.
See User Names, Passwords, and Groups for more information.

Delete Entry / Delete Group

When an entry is selected, this will delete it. When a Group is selected (in Tree view), this will delete all entries under the selected group. Confirmation is requested if there is more than one entry in the selected group. This is disabled if the database is opened read-only.

Rename Entry / Rename Group

Rename the selected entry (or group, in Tree view). This is disabled if the database is opened read-only.

Find Entry

Find an entry containing a given text. By default, all fields except the Password field are searched.
See Searching for more information.

Duplicate Entry

Duplicates the selected entry. This is disabled if the database is opened read-only.

Add Group

Create a new password group (e.g., Banking Passwords). This is disabled if the database is opened read-only.

Clear Clipboard

Clear the clipboard's memory. This clears the clipboard of anything that was pasted there by Password Safe  (password, username, notes, etc.). If the current contents of the clipboard did not come from Password Safe, then this will have no effect.

Copy Password to Clipboard

Copy the current selection's password to the clipboard.

Copy User Name to Clipboard

Copy current selection's user name to the clipboard.

Copy Notes to Clipboard

Copy the contents of current selection's notes field to the clipboard.

Copy URL to Clipboard

Copy the current selection'URL field to the clipboard.

Browse to URL

When an entry is selected, and the entry has a URL associated with it, then this will invoke the default browser to the URL. If no entry is selected, or the selected entry's URL is blank, this command will be disabled.

Perform Auto Type

Automatically input user name and password into the selected input field.
See Using Auto Type for more information.

Create Shortcut

When an entry is selected, this allows you to create a reference to the entry, typically in another group. This is useful if you have an entry that "belongs" under more than one group, but you don't wish to duplicate data. A shortcut has a distinct icon associated with it, , and by default has the name "shortcut to x", where x is the title of the original entry. Of course, you can edit the name, move or delete the shortcut, just like a regular entry.

Edit by right mouse click

Entry Alias Group

A lot of functions in the Edit menu can be reached also by clicking the right mouse button on the corresponding entry or group. Here will be descibed only those functions that are not mentioned above in chapter Edit Menu.

Display subset of Password

In the upper field you can define the positions of the password characters you want for the subset of the password. You can use spaces, commas or semi-colons to separate the positions. In the lower field you will see the corresponding characters of the password. To quit, please use the Esc key.

Go to Base entry

This item will only appear for a shortcut or alias entry. After clicking Go to Base entry the corresponding base entry will be choosen.
