Password Safe and YubiKey

YubiKey is an innovative hardware authentication device developed and sold by Yubico. To the user, it looks like a small, thin, disk-on-key with a button on top. To PasswordSafe, it functions as an authentication token, providing strong physical security to the password database. Here we describe the steps needed to enable YubiKey authentication for PasswordSafe.

Getting YubiKey

Obviously, you need a YubiKey device to start. YubiKeys can be purchased online from here. Note that PasswordSafe requires YubiKeys with firmware version 2.2 and above.

Installing the Windows Client API

For PasswordSafe to communicate with your YubiKey, the Windows Client API needs to be installed on your PC. You can download the binary installer for the Windows Client API free of charge from here. After downloading, just double-click on the downloaded file and follow instructions. Note that without the client API, this version of PasswordSafe cannot recognize your YubiKey, and will only offer Password-based authentication.

Configuring your YubiKey

Your YubiKey needs to be configured properly in order to work with PasswordSafe. Yubico provides several personalization tools that can be used to configure YubiKey devices here. The following instructions refer to the Windows Configuration Utility.
  1. Start the YubiKey Personalization Tool
  2. Insert your YubiKey into a free USB slot on your PC
  3. From the top bar of tha application window, click on Challenge-Response.
  4. Click on Configuration Slot 2
  5. Check "Require user input (button press)"
  6. Select HMAC-SHA1 Mode "Fixed 64 byte input"
  7. Click on the Generate button to create a unique Secret Key
  8. Click on Write Configuration and make sure that the status in the Results window shows "YubiKey has been successfully configured"
  9. Your YubiKey is now ready to be used with PasswordSafe
  10. Instructions for using your YubiKey to access a Password Safe database may be found in Creating a New Safe.
Note: The Secret Key generated in step 7 is what makes your YubiKey unique. If you wish, you can create a spare YubiKey with the same secret key to use in case of loss of the original YubiKey. Just remove the original YubiKey, insert the spare, and click on Write Configuration without changing the Secret Key.
