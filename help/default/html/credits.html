Program Credits

Grateful thanks to Bruce Schneier for creating the Twofish algorithm and for releasing Password Safe to the public; and to SourceForge.net for hosting the project and supporting the Open Source community.
Password Safe version 3.29 and later use pugixml by Arseny Kkapoulkine for parsing the application's configuration file. The code for pugixml is provided under the MIT license.
Password Safe version 3.12 and later uses an algorithm created by Tom Van Vleck, Morrie Gasser, and Dan Edwards to generate pronounceable passwords.

Thanks to other developers that have published their code for the benefit of other projects:

  • Daniel Lohmann's method for ensuring only one instance of an application can run at any time. See http://www.netexec.de/
  • CInfoDisplay to replace MS's broken ToolTips support and based on a class of the same name in Asynch Explorer by Joseph M. Newcomer [MVP]. See http://www.flounder.com
  • Explorer view in a Dialog Based Application based on code submitted by Gerd Klevesaat to Codeguru on 11 May 1998. See http://www.codeguru.com/article.php/c1979
  • The hiding and showing of rows in the Explorer view based on Kirk Stowell's extension of a similar class written by Oleg Galkin published on 7 August 1998. See http://www.codeguru.com/Cpp/W-D/splitter/article.php/c1543 with Kirk's extension from within the legacy post in the comments made on 28 December 2000.
  • The flat ComboBox used on the Explorer toolbar based on XTFlatComboBox also by Kirk Stowell. See http://www.codejock.com/support/articles/mfc/controls/con_1.asp
