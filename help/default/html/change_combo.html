Changing Master Passwords

To change a master password ("Safe Combination"), first open up the database whose password you wish to change. Once opened, go to the Manage menu and select Change Safe Combination. You should enter the current master password, and then the new master password (twice, for confirmation). 

This customized version of Password Safe enforces the following policy for master passwords:

  1. A master password must have at least three out of the following types of characters:
    1. Lower case
    2. Uppercase
    3. Digits
    4. Symbols
  2. A master password must be at least 15 characters long.
  3. The master password must be changed at least once a year.
Notes:
  • There is no mechanism within Password Safe to retrieve lost or forgotten safe passwords.
    This is a deliberate design decision, since any mechanism for retrieving lost passwords poses a security hole that can be exploited.
  • Please choose a difficult-to-guess password. A trivial password, such as "password" or your name is easy to guess, and negates the security that Password Safe provides.
