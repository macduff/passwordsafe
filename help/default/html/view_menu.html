View Menu

The View Menu controls the available display options. You can switch between how your entries' user names and passwords are organized: either as a single list (Flattened List) or organized by category (Nested Tree). You can choose whether or not to display a toolbar, and if so, if to use the Classic or New Toolbar Icons. The toolbar itself is customizable. You can also expand or collapse all the groups displayed in the Nested Tree view, filter the entries displayed to match any criteria, select the font in which the entries are displayed and display reports of merge, compare, import and other advanced operations.

Switching Between Nested Tree and Flattened List Views

  1. Go to the View menu
  2. Select Nested Tree or Flattened List

Controlling the Toolbars

To display or hide the toolbar, check or uncheck the Toolbar Visible menu item.
To use "new" or "classic" toolbar icons, select the "New toolbar" or "Classic toolbar" menu item, respectively.

The "Dragbar" is a special toolbar that allows you to paste parts of the currently selected entry by "dragging" the icon specifying the part you want to paste to the window that you want it to be pasted to. For example, to paste the "Notes" text of a given entry to another application using the dragbar, select the entry, and then drag the "N" icon from the dragbar to the application. When you release the left mouse button, the contents of the Notes field will be pasted to the application. The dragbar will disable ("grey out") icons for fields that are empty for the selected entry. The 'Autotype' image is slightly different to the others in that dragging the image to an input area on another Window (e.g. Browser) will cause Password Safe to begin Autotype from there.

To display or hide the dragbar, check or uncheck the Dragbar Visible menu item.

Expand All

Expands all groups, making all entries visible (Nested Tree view only).

Collapse All

Collapses all groups (Nested Tree view only).

Refresh F5

Refreshes the display either by pressing the F5 key or clicking on this menu item.

Show entries with Expiry dates

If you select this menu item, only entries with an expiry date will be shown in the list. It does not matter that the expiry date is in the past or future.

If there aren't any entries with expiry date, this menu item will be grayed out.

Filters

A "filter" is a criteria (or set of criteria) that an entry either matches or doesn't match. Password Safe allows you to define filters, and display only those entries that matches a given filter. For more information on defining and using filters, see here.

Change Font

Select a different font for displaying entries in the List or Tree views, and for displaying passwords in the Add and Edit Entry dialog boxes.

Reports

View the results of merge, compare, import and validate operations.
