Manage Menu

The Manage Menu is where all of Password Safe's options and settings are controlled.

Change Safe Combination

Selecting this option brings up the Change Safe Password dialog.

See Changing Master Passwords for more information.

Make Backup

Create a backup copy of the current password database.

See Backing Up and Restoring Databases for more information.

Restore from Backup

Restore a backed up password database file.

See Backing Up and Restoring Databases for more information.

Options...

Selecting Options... brings up a tabbed dialog where many of Password Safe's settings can be viewed and edited. To learn more about a specific tab select it below:

Backups Tab
Display Tab
Misc Tab
Password History Tab
Security Tab
Shortcuts Tab System Tab

Note: The Password Policy Tab has been superceded by the following menu item to manage the database default policy and named password policies.

Note:
Many of the setting are stored as preferences, as described here.

Password Policies...

This allows you to manage both the database default password policy and named password policies. You may create up to 255 named password policies and the name of each of these policies cannot be greater than 255 characters.

Validate

After selecting a database, this database will be checked for being completely compatible to Password Safe.

Note: This menu item will only be enabled if there is no database currently opened.

Customize Main Toolbar

This allows you to select which commands appear on the toolbar, and in which order.

Generate Password

This allows you to generate a password with a specified password policy to be put into the clipboard, by pushing Generate. The password in the clipboard can then be used by some other application. You may select the database default password policy, a named password policy or use a specific policy you enter into dialog.

Change Language

If you have installed all the currently available additional language DLLs when Password Safe was installed (which is the default), you will have the ability to switch between these languages on-the-fly via this menu item.

After changing the language, all menus, dialogs and messages will now be in your chosen language.
