Password Safe

你有多少用户名和密码需要跟踪呢？无论答案是一个或数百个，Password Safe 能够让你轻松、安全地建立一个安全并加密的主密码列表，以及提供一个单独的主密码解除所有锁定的列表。

安全始于用户！保留书面密码列表在便条纸或在桌面文本文档是极为不安全的，并很容易被窥探（基于网络和人两者）。使用相同密码一遍又一遍的在广泛的系统和网站产生的恶梦般的情况，要是一旦有人算出一个密码，他们将能算出你的所有密码，并立刻有机会获得你生活中的每一部份（系统、电子邮件、零售、财务、工作）。

Password Safe 允许用户管理旧密码和轻松快捷的生成、存储、组织、检索及使用复杂的新密码；利用密码策略控制，一旦存储，每次仅需点击几下用户名和密码便可离开。

利用 Password Safe 用户能够使用自己的自定义参考组织密码，例如：用户ID、类别、网站或位置。并可以选择存储所有密码在单独加密的主密码列表(一个加密的密码数据库)中或使用多个数据库进一步组织密码(例如：工作和家庭)，以及它具有直观的界面和快速运行的优点。
