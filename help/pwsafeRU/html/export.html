Экспорт элементов

Password Safe поддерживает экспорт и импорт сохранённых паролей из различных форматов. Для экспорта данных следует использовать пункт меню «Файл»→«Экспортировать в». Поддерживаются следующие форматы:

  • Password Safe v1 (для использования в Password Safe версии 1.x)
  • Password Safe v2 (для использования в Password Safe версии 2.x)
  • Простой текст
  • XML

Для экспорта в форматы предыдущих версий Password Safe просто выберите нужный формат и задайте имя контейнера.

Чтобы экспортировать элементы в другой контейнер Password Safe, запустите ещё одну копию Password Safe (должен быть разрешён запуск нескольких экземпляров на вкладке «Управление→Настройки→Система») и создайте новый контейнер. Затем перетяните необходимые элементы или группы в новый контейнер.

Экспорт в текст

Примечание. Экспорт в текстовый формат приводит к созданию незащищённой копии указанных элементов. Простого удаления файла после использования не достаточно для предотвращения доступа злоумышленников к его содержимому, что может быть использовано для обхода защиты, предоставляемой Password Safe.

Password Safe позволяет произвести экспорт элементов в текстовый формат. Затем эти файлы могут быть использованы или изменены другими приложениями.
Для подтверждения экспорта данных в простой текст будет запрошен мастер-пароль.

Также можно указать дополнительные параметры:

  • Поля экспортированных записей («Группа.Заголовок», «Имя пользователя» и «Заметки») будут разделены символом табуляции. «Группа.Заголовок» считается одним полем. Любое значение, содержащее пробел, будет заключено в двойные кавычки. Только поле «Заметки» может состоять из нескольких строк. Обычно, каждая строка заметок записывается в отдельной строке. Но, в данном случае, затрудняется сортировка и сравнение экспортированных файлов. Для упрощения сортировки вы можете указать разделитель, который будет вставляться между строками. Использованный разделитель нужно будет указать при импорте, иначе текст заметок не будет разбит на строки.

    Если требуется, укажите символ для разделения строк с заметками. Так как всё содержимое поля «Заметки» заключено в двойные кавычки, это символ нельзя использовать в качестве разделителя строк.

Примечание. Описание формата экспорта в текстовый файл находится в разделе Формат текстового экспорта и импорта.

XML

XML (eXtensible Markup Language) — распространённый формат для передачи данных между приложениями. Password Safe может экспортировать и импортировать содержимое контейнера в данный формат. Структура (схема) для экспорта и проверки документа определена в файле pwsafe.xsd, который идёт в комплекте с Password Safe. Этот файл используется Password Safe при импорте данных из формата XML и может потребоваться приложениями, которые будут обрабатывать экспортированные данные.

Примечание. Экспорт в текстовый формат приводит к созданию незащищённой копии указанных элементов. Простого удаления файла после использования не достаточно для предотвращения доступа злоумышленников к его содержимому, что может быть использовано для обхода защиты, предоставляемой Password Safe.

Отображение XML-файлов в браузере описывается в стилевом файле pwsafe.xsl (XSL — eXtensible Stylesheet Language). Файл, идущий в комплекте с Password Safe, является примером оформления и может быть изменён под ваши нужды. Так как стилевой файл лишь задаёт отображение данных, а не изменяет их, его содержимое не влияет на возможность импорта из XML-файла в контейнер Password Safe.
