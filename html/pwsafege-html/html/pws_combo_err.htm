Falsches Datenbank-Passwort

Das Fenster mit dem Titel "Incorrect Passkey" erscheint, wenn Sie das Passwort für die Datenbank dreimal hintereinander falsch eingegeben haben. Falls Sie sich bei der Eingabe nur vertippt haben, können Sie es noch einmal versuchen (Try Again). Sie können auch eine andere Datenbank öffnen (Open Another Database...) oder eine komplett neue Datenbank erstellen (Create a New Database).

Falls Sie das Passwort für die Datenbank vergessen haben sollten, stellen Sie das Öffnen der Datenbank erst einmal zurück. Vielleicht fällt Ihnen das richtige Passwort später irgendwann wieder ein. Leider kann Ihnen nicht geholfen werden, wenn Sie das Passwort vergessen haben. Die Sicherheit dieses Programms beruht nämlich auf der Tatsache, dass die Datenbank ohne Kenntnis des richtigen Passwortes nicht geöffnet werden kann.
