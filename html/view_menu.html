View Menu

The View Menu controls the available display formats. Users can switch between Classic and New Toolbar Icons as well as how their user names and passwords are organized: either as a single list (Flattened List Mode) or organized by category (Nested Tree Mode).

Switching Between Menu Icon Views

  1. Go the the View menu
  2. Select Classic Toolbar or New Toolbar

Switching Between Nested Tree and Flattened List Modes

  1. Go the the View menu
  2. Select Nested Tree Mode or Flattened List Mode
